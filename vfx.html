Visual Effects Library

This library is an ongoing repository for various Visual Effects that I've created since getting into it about a year ago. So far some have appeared in games that I've made, but I've also made some for other games. Most notably, I had a few different effects such as the Plasma Torch effect incorporated in the game Endothermic; a MSU student project that partnered with members from the Deep Silver Volition games company.
